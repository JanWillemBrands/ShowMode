The ShowMode app is a kiosk style app that needs to run on a single ioPad mini 3 running iOS 12.5.8.
It will not need to be published on the AppStore.
The app will run in full-screen portrait mode at all times.
The background is a full-screen rotating photograph of beautiful nature shots.
The foreground displays time & weather in Padua, Italy.  It also displays an RSS feed of BBC world headline news, using pictures as well as text.
The layout style and fonts should mimic the styles used by DAKboard, or Amazon Fire Show Mode.
